This repository contains **vcpkg portfiles** for the [`sparkplug`](https://github.com/JLanda91/sparkplug) project.  
It allows consumers to easily install and use `sparkplug` as a dependency
through [vcpkg](https://github.com/microsoft/vcpkg).

## About Sparkplug
- **Project:** [JLanda91/sparkplug](https://github.com/JLanda91/sparkplug)
- **License:** Apache-2.0
- **Author:** Jasper Landa
- **Description:** a testing and mocking framework for CUDA device functors built on gtest.
